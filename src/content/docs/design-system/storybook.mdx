# Storybook

The design system component gallery lives in [`manovaspace/design-system`](https://github.com/manovaspace/design-system), not in this docs repo.

## Run locally

```bash
git clone https://github.com/manovaspace/design-system.git
cd design-system
pnpm install
pnpm storybook
```

Runs on port **10006**.

## Manova workspace

If you have the full Manova workspace:

```bash
cd manovaspace/design-system
pnpm storybook
```

Or from `orbit/orbit-frontend`:

```bash
pnpm storybook
```

(Delegates to the design-system repo.)

## What it covers

Stories for `@manovaspace/ui` primitives, motion, tokens, and theme switching — useful when developing components without a full Next.js app.
