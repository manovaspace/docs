MIT design tokens and Radix/Tailwind UI primitives published as `@manovaspace/tokens`, `@manovaspace/ui`, and `@manovaspace/devtools`.

## Packages

| Package | Use when |
| --- | --- |
| [`@manovaspace/tokens`](/docs/design-system/packages/tokens/) | You need CSS variables, fonts, and token JSON |
| [`@manovaspace/ui`](/docs/design-system/packages/ui/) | You build screens with shared components |
| [`@manovaspace/devtools`](/docs/design-system/packages/devtools/) | You want a **dev-only** token/theme toolbar |

## Storybook

Browse components locally — [Storybook](/docs/design-system/storybook/) (port 10006).
