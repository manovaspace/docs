## Install

```bash
pnpm add @manovaspace/tokens @manovaspace/ui
```

Peer dependencies: `react` and `react-dom` ^19.2. Also works with npm or yarn.

## Styles

In your global CSS:

```css
@import "@manovaspace/tokens/fonts.css";
@import "@manovaspace/tokens/tokens.css";
```

```tsx
import "@manovaspace/ui/styles.css";
```

## Hello world

```tsx
import { Button } from "@manovaspace/ui";

export function Example() {
  return <Button>Get started</Button>;
}
```
