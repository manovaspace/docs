Optional **development-only** toolbar for editing tokens and previewing theme changes. Do **not** ship this package in production client bundles.

:::note[Legacy identifiers]
Some exports still use `Orbit*` names (for example `OrbitDevtoolsProvider`). The npm package is `@manovaspace/devtools`.
:::

## Install

```bash
pnpm add -D @manovaspace/devtools
```

Requires `@manovaspace/tokens` and `@manovaspace/ui`. Also works with npm or yarn.

## Usage

```tsx
import {
  OrbitDevtoolsProvider,
  DevToolsShell,
} from "@manovaspace/devtools";

export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <OrbitDevtoolsProvider config={{ appName: "my-app" }}>
      <DevToolsShell>{children}</DevToolsShell>
    </OrbitDevtoolsProvider>
  );
}
```

```css
@import "@manovaspace/devtools/styles.css";
```

For production-safe slot registration (stub in prod builds):

```tsx
import { DevtoolsSlot } from "@manovaspace/devtools/slot";
```

## Exports

| Import | Purpose |
| --- | --- |
| `@manovaspace/devtools` | `OrbitDevtoolsProvider`, `DevToolsShell`, save helpers |
| `@manovaspace/devtools/slot` | `DevtoolsSlot` — prod-safe stub |
| `@manovaspace/devtools/next` | Theme route handlers for Next.js |
| `@manovaspace/devtools/styles.css` | Toolbar styles |
