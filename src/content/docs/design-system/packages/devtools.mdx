# @manovaspace/devtools

Development-only toolbar for editing design tokens and previewing theme changes. **Do not import in production bundles.**

## Install

```bash
npm install -D @manovaspace/devtools
```

Requires `@manovaspace/tokens` and `@manovaspace/ui`.

## Usage

```tsx
import {
  OrbitDevtoolsProvider,
  DevToolsShell,
} from "@manovaspace/devtools";

export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <OrbitDevtoolsProvider
      config={{
        appName: "my-app",
      }}
    >
      <DevToolsShell>{children}</DevToolsShell>
    </OrbitDevtoolsProvider>
  );
}
```

```css
@import "@manovaspace/devtools/styles.css";
```

For production-safe slot registration (zero overhead in prod builds), use `@manovaspace/devtools/slot`:

```tsx
import { DevtoolsSlot } from "@manovaspace/devtools/slot";
```

## Exports

| Import | Purpose |
| --- | --- |
| `@manovaspace/devtools` | `OrbitDevtoolsProvider`, `DevToolsShell`, save helpers |
| `@manovaspace/devtools/slot` | `DevtoolsSlot` — production-safe stub |
| `@manovaspace/devtools/next` | Next.js theme route handlers |
| `@manovaspace/devtools/styles.css` | Devtools stylesheet |
