# @manovaspace/tokens

Design tokens for the manovaspace UI layer: default token JSON, generated CSS variables, and font assets used by `@manovaspace/ui`.

## Install

```bash
npm install @manovaspace/tokens
```

## CSS

```css
@import "@manovaspace/tokens/tokens.css";
@import "@manovaspace/tokens/fonts.css";
```

## Exports

| Import | Purpose |
| --- | --- |
| `@manovaspace/tokens` | Token types and runtime helpers |
| `@manovaspace/tokens/tokens.css` | Semantic CSS variables |
| `@manovaspace/tokens/fonts.css` | Font face declarations |
