Token JSON, generated CSS variables, and font assets used by [`@manovaspace/ui`](/docs/design-system/packages/ui/).

## Install

```bash
pnpm add @manovaspace/tokens
```

Also works with npm or yarn.

## CSS

```css
@import "@manovaspace/tokens/fonts.css";
@import "@manovaspace/tokens/tokens.css";
```

Semantic CSS variables power theming (background, foreground, primary, muted, border, and color families with steps **50 → 950**).

## Runtime helpers

```ts
import {
  defaultTokensRaw,
  resolveTokens,
  applyTokens,
} from "@manovaspace/tokens";
```

Use `TokenProvider` from `@manovaspace/ui` for React apps that merge defaults with overrides.

:::note[Legacy identifiers]
Some TypeScript types still use `Orbit*` names (for example `OrbitTokensRaw`). The npm package is `@manovaspace/tokens`.
:::

## Exports

| Import | Purpose |
| --- | --- |
| `@manovaspace/tokens` | Types, `defaultTokensRaw`, resolve/apply helpers |
| `@manovaspace/tokens/tokens.css` | Semantic CSS variables |
| `@manovaspace/tokens/fonts.css` | Font faces |
