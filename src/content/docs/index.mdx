## Who this is for

Teams building **Next.js / React** apps that want reusable MIT packages for TypeScript config, markdown, PWA wiring, and UI primitives — without forking a private toolkit.

## Install paths

**Design system** (tokens + components):

```bash
pnpm add @manovaspace/tokens @manovaspace/ui
```

**Utilities** (pick what you need):

```bash
pnpm add -D @manovaspace/tsconfig
pnpm add @manovaspace/markdown @manovaspace/pwa @manovaspace/observability
```

Also works with npm or yarn.
