Shared TypeScript utilities published as `@manovaspace/*` on npm. Use them independently or alongside the [design system](/docs/design-system/).

## Packages

| Package | Use when |
| --- | --- |
| [`@manovaspace/tsconfig`](/docs/utilities/packages/tsconfig/) | You want strict shared TypeScript presets |
| [`@manovaspace/markdown`](/docs/utilities/packages/markdown/) | You need GFM markdown in React |
| [`@manovaspace/pwa`](/docs/utilities/packages/pwa/) | You are adding a Next.js PWA / offline shell |
| [`@manovaspace/observability`](/docs/utilities/packages/observability/) | You wire Sentry via Next.js `instrumentation.ts` |

[Getting started](/docs/utilities/getting-started/) — install from npm and pick the packages you need.
