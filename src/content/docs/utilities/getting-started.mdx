`@manovaspace/*` resolves from [registry.npmjs.org](https://registry.npmjs.org). No special `.npmrc` is required for these packages.

## Install

```bash
pnpm add -D @manovaspace/tsconfig
pnpm add @manovaspace/markdown
pnpm add @manovaspace/pwa
pnpm add @manovaspace/observability
```

Also works with npm or yarn — examples use **pnpm**.

## What to use

| Need | Package |
| --- | --- |
| Shared TypeScript presets | [tsconfig](/docs/utilities/packages/tsconfig/) |
| GFM markdown in React | [markdown](/docs/utilities/packages/markdown/) |
| PWA / offline | [pwa](/docs/utilities/packages/pwa/) |
| Sentry via `instrumentation.ts` | [observability](/docs/utilities/packages/observability/) |
